# Оформление для каталога и сниппетов VK

## Иконки

**Универсальная иконка** (каталог, лента, сообщения, рекомендации VK):
- **Файл:** `public/icon-576.png`
- **Размер:** 576×576 px
- **Формат:** PNG

**Маленькая иконка:** `public/icon-small.png` (48×48), также есть `icon-small.svg`.

Исходники (SVG): `public/icon.svg`, `public/icon-small.svg`. Чтобы перегнать в PNG заново:
```bash
npm run icons:png
```
Скрипт создаёт: `icon-576.png` (576×576), `icon-small.png` (48×48), `icon-128.png` (128×128).

В `index.html` подключены PNG для favicon и apple-touch-icon. В кабинет VK загружайте **icon-576.png** как универсальную иконку.

## Скриншоты для экрана запуска (десктоп VK)

**Требования VK:** размер **1200 × 600 px**, формат JPG/PNG/GIF, ориентация **альбомная**. Рекомендуют не меньше 3 штук.

**Как не возиться с размерами:**
1. Сделайте скриншоты приложения (главный экран, результат, история) в любом размере и сохраните в папку **`docs/screenshots/`** (например `01-main.png`, `02-result.png`).
2. В корне проекта выполните:
   ```bash
   npm run screenshots:vk
   ```
3. Готовые картинки 1200×600 появятся в **`docs/screenshots/vk-1200x600/`** — их и загружайте в кабинет VK в блок «Скриншоты».

Скрипт обрезает изображения по центру до 1200×600, так что кадрировать вручную не нужно.

## Кнопка «Добавить из друзей VK»

Если при нажатии появляется ошибка «Не удалось загрузить друзей»:

1. **Открыто не в VK** — друзья подгружаются только когда приложение открыто **внутри ВКонтакте** (мобильное приложение VK или сайт m.vk.com). По прямой ссылке в обычном браузере список друзей недоступен — приложение так и сообщит.
2. **Запрашиваемые процедуры** — в настройках мини-приложения на [dev.vk.com](https://dev.vk.com/) может быть блок **«Запрашиваемые процедуры»** (или похожее название). Если он есть, добавьте туда процедуру, связанную с доступом к друзьям: в списке ищите **«Друзья»**, **friends** или **friends.get**, включите и сохраните настройки. После этого снова откройте приложение из VK и попробуйте «Добавить из друзей VK».
3. **Если всё равно не грузится** — добавление участников **вручную** (ввод имён) всегда доступно и достаточно для жеребьёвки.

Участников всегда можно добавить вручную — этого достаточно для работы жеребьёвки.

## Текст для описания в каталоге

**Краткое описание (сниппет):**
Случайный выбор: кто платит за кофе, кто выбирает фильм, кто за рулём. Добавьте друзей из VK или введите имена — жеребьёвка в один тап.

**Полное описание (при необходимости):**
Мини-приложение «Кто платит?» помогает случайно выбрать человека для любого сценария: кто платит за кофе, кто выбирает фильм, кто за рулём, кто дежурит, кто заказывает еду, кто ставит музыку — или свой вариант. Участников можно вводить вручную или подтянуть из друзей VK. Результат можно отправить в переписку или на стену. История последних выборов сохраняется в приложении.
