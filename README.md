# Кто платит?

Мини-приложение ВКонтакте для компании: помогает случайным образом выбрать, кто платит за кофе, кто выбирает фильм, кто за рулём и т.п.

## Что умеет

- **Сценарии** — встроенные (кофе, фильм, руль, дежурный, заказ еды, музыка) или свой вопрос.
- **Участники** — можно вводить имена вручную или подтянуть друзей из ВКонтакте.
- **Жеребьёвка** — по кнопке случайно выбирается один человек, результат можно отправить в VK.
- **История** — сохраняются последние выборы (в браузере).

## Как запустить локально

```bash
npm install
npm run dev
```

В браузере откроется адрес вида `http://localhost:5173`. Полноценно друзья и шеринг работают внутри VK — после публикации приложения.

## Сборка

```bash
npm run build
```

Собранные файлы лежат в `dist/`. Для публикации в VK укажите свой `app_id` в `vk-hosting-config.json` и залейте содержимое `dist` на хостинг.

## Стек

React, TypeScript, Vite, VKUI, VK Bridge.
